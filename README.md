## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone [https://github.com/your-username/backend-learning.git](https://github.com/Raju269/Backend-Work.git)
cd backend-learning
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file in root:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

---

### 4️⃣ Run the Server

```bash
npm run dev
```

or

```bash
node server.js
```
